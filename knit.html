Hi! I made this so I could test out intarsia knitting patterns. If you're using it I hope it helps you too!

Click on stitches to add colour, number keys swap between colours, f+click to fill, shift+click for straight lines, z to undo. This runs best on desktop/laptop but you can still try on your phone.

If you have any issues or find any bugs send me a message @olandobrando on Instagram or TikTok.

If you make anything I would love to see! tag me/dm @olandobrando
